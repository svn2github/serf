Issue Tracker

Apache Serf tracks both bugs and enhancements using JIRA issue tracker. Before filing new requests, we ask that you first do the following:

  • Search the JIRA database
  • Check the project mailing list via searching the archives or asking a question

Source Control

Serf's source code is stored in the serf tree of the main Subversion repository of the Apache Software Foundation. You can checkout the latest version of the code using the following command: 

% svn checkout https://svn.apache.org/repos/asf/serf/trunk serf-trunk
